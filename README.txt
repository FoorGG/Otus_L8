Необходимо скачать скрипт и его запустить, далее следовать по инструкции внутри скрипта.
Скрипт создаст zpool с именем mypool.
Скрипт автоматически находит все диски которые добавлены были, и находит все диски которые не примонтированы и не имеют разделов.
Скрипт запускается 1 раз, после успешного выполнения его использовать повторно не надо.
